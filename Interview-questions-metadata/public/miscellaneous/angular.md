# Some remembrables
- **To run angular application and open in default browser**
    ```
    ng serve -o
    ```
- **To generate component without test file and css file**
    ```
    ng generate component --spec=false --is
    ```
- **Building blocks of angular**
    - Modules
    - Components
    - Metadata
